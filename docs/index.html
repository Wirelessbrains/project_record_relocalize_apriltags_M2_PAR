Record, Map, and Relocalize with AprilTags

A practical pipeline to learn a trajectory, rebuild references offline, and measure online deviation from the learned path.

Quick Start See Results

What This System Does

Offline Build

Extract trajectory and map artifacts from rosbag datasets.

Online Relocalization

Compute nearest-point distance and heading error in real time.

Simulation Demo

Validate end-to-end behavior and control integration before field tests.

Pipeline

  1. Record telemetry and tag detections.
  2. Build reference trajectory and tag map offline.
  3. Relocalize live robot pose against learned trajectory.
  4. Evaluate deviation metrics and visualization outputs.

Practical Work Example

Suggested student exercise:

  2. Generate reference trajectory and map outputs.
